## String

String é uma sequência de caracteres que podem ser declaradas de três formas:

Com aspas simples:

```javascript
var hello = 'Olá'; 
```

Com aspas duplas:
```javascript
var hello = "Olá"; 
```

Com crase:
```javascript
var hello = `Olá`; 
```

## Number

Em JavaScript qualquer número é considerado um **number**:

Exemplo inteiro:

```javascript
var numInt = 5;
```

Exemplo decimal:

Exemplo **NUMBER**:

```javascript
var numDec = 1.25; 
```
